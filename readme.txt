== Description ==

The plugin helps you set points on the map and filter them in the future.
The plugin is based on the Yandex.Maps API.
You can put a map on your website using a shortcode.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'.
2. Search for 'Many points on the map'.
3. Activate the plugin from your Plugins page.
4. Set the desired points on the map.

= From WordPress.org =

1. Download 'Many points on the map'.
2. Upload the 'Many points on the map' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Many points on the map' from your Plugins page.
